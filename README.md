# tas.skoudros.com

One-page profile site for Tas Skoudros, built with Vite, React, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Edit the primary copy and links in `src/data/content.ts`.

## Production

```bash
npm run build
npm run preview
```

The generated `dist` directory can be deployed to Netlify, Cloudflare Pages, or Vercel. Use `npm run build` as the build command and `dist` as the publish directory.
